# 全域開發規範

用 Google Apps Script（GAS）+ JavaScript 來開一門 54 小時的應用型課程,它能讓學生一邊學 JavaScript 語法，一邊看到真實自動化與雲端整合的效果。

## 課程大綱

1. 認識 Apps Script 與 JavaScript 基礎
2. 操作 Google Sheets
3. 自動寄信與通知
4. 檔案與雲端操作
5. 表單整合
6. 建立簡易 Web App
7. API 整合應用
8. 綜合實作展示

## 實作案例建議（可搭配上表單元)

| 編號  | 實作主題                | 重點技能                     |
| --- | ------------------- | ------------------------ |
| 1️⃣ | 學生成績登錄系統            | 操作 Google Sheets         |
| 2️⃣ | 自動寄送通知信             | MailApp + Trigger        |
| 3️⃣ | Google Form 自動收集與統計 | FormApp + Sheets 整合      |
| 4️⃣ | 雲端硬碟整理助手            | DriveApp 操作檔案與資料夾        |
| 5️⃣ | 簡易簽到系統              | HTML Service + Sheets    |
| 6️⃣ | 自動產出成績單 PDF         | DriveApp + Docs Template |
| 7️⃣ | 天氣查詢 Web App        | 外部 API + JSON 處理         |
| 8️⃣ | 每日報表寄送機器人           | 定時觸發器 + 資料摘要             |
| 9️⃣ | 專案檔案上傳管理工具          | HTML 前端 + Drive API      |
| 🔟  | 綜合專案：校內服務申請系統       | 全功能整合應用                  |



## 課程特色（適合學校或進修中心開課)

💬 強調「學語法 + 做產品」並重。

🧩 可結合 Google 生態系，學生容易實作。

🚀 結束後能製作自動化或小型 Web App。

👨‍🏫 適合你這類「電腦與應用導向」課程。
