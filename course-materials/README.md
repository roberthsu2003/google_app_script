# Google Apps Script + JavaScript 應用型課程教材

本資料夾包含完整的 54 小時 GAS + JavaScript 課程教材。

## 課程結構

- **unit-01-javascript-basics/** - 認識 Apps Script 與 JavaScript 基礎（10 小時）
- **unit-02-google-sheets/** - 操作 Google Sheets（6 小時）
- **unit-03-email-notifications/** - 自動寄信與通知（4 小時）
- **unit-04-files-drive/** - 檔案與雲端操作（6 小時）
- **unit-05-forms-integration/** - 表單整合（4 小時）
- **unit-06-web-apps/** - 建立簡易 Web App（8 小時）
- **unit-07-api-integration/** - API 整合應用（6 小時）
- **unit-08-final-project/** - 綜合實作展示（10 小時）
- **shared-resources/** - 共用資源與參考文件

## 使用說明

1. 按照單元順序循序漸進學習
2. 完成每個單元的練習題後再進入下一單元
3. 每個實作案例都要親自動手完成
4. 參考 shared-resources 中的語法支援對照表

## 實作案例總覽

本課程包含 10 個完整實作案例，涵蓋 Google Workspace 生態系統的各項應用。

## 教學者注意事項

- 請先閱讀 shared-resources/instructor-guide.md
- 確認所有範本檔案已正確設定
- 準備測試用的 Google 帳號與資料
