# 共用資源

本資料夾包含課程中所有單元共用的參考文件和資源。

## 文件清單

### 1. GAS JavaScript 語法支援對照表
**檔案：** `gas-javascript-syntax-support.md`

完整的 JavaScript 語法支援對照表，標註每個語法在 GAS 中的支援情況：
- ✅ 完全支援（無需 V8 runtime）
- ⚠️ 需要 V8 Runtime
- ❌ 不支援

包含：
- 變數宣告
- 資料型別
- 運算子
- 流程控制
- 迴圈
- 函式
- 陣列方法
- 物件操作
- 字串處理
- JSON 處理
- 錯誤處理
- 除錯工具
- 類別與物件導向
- Promise 與非同步
- 正規表達式

### 2. 範本規格說明
**檔案：** `template-specifications.md`

詳細說明課程中使用的所有範本檔案結構：
- 學生成績試算表範本
- 通知名單試算表範本
- 表單回應試算表範本
- 簽到記錄試算表範本
- 服務申請系統試算表範本
- Google Docs 成績單範本
- Google Form 範本
- 測試資料準備指引

### 3. 教學者指引
**檔案：** `instructor-guide.md`

完整的教學指引文件：
- 課程準備檢查清單
- 教學節奏建議
- 18 週課程進度表
- 評量方式建議
- 常見教學挑戰與解決方案
- 線上實體混合教學指引

### 4. 測試帳號設定指引
**檔案：** `test-account-setup.md`

測試環境準備的完整說明：
- 測試帳號規劃
- 帳號建立步驟
- 範本檔案建立
- 測試資料準備
- 權限設定
- 檔案 ID 記錄

## 使用建議

### 給教學者

1. **開課前必讀**
   - instructor-guide.md（教學指引）
   - test-account-setup.md（測試環境準備）

2. **教學時參考**
   - gas-javascript-syntax-support.md（語法支援對照）
   - template-specifications.md（範本規格）

3. **分享給學習者**
   - gas-javascript-syntax-support.md（語法參考）
   - 範本檔案清單（從 test-account-setup.md 整理）

### 給學習者

1. **學習時參考**
   - gas-javascript-syntax-support.md
   - 查詢語法支援情況
   - 選擇適當的寫法

2. **實作時使用**
   - 複製範本檔案
   - 參考範本規格
   - 準備測試資料
