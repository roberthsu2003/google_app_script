## 測試帳號規劃

### 建議的測試帳號配置

#### 1. 教師帳號（管理者）× 1

**用途：**
- 建立所有範本檔案
- 設定檔案權限
- 測試管理功能
- 審核流程測試

**帳號資訊範例：**
```
Email: teacher.gas.course@gmail.com
姓名: 張老師
角色: 教師/管理者
```

#### 2. 學生帳號（一般使用者）× 3-5

**用途：**
- 測試學生端功能
- 表單提交測試
- 權限驗證
- 多人協作測試

**帳號資訊範例：**
```
Email: student1.gas.course@gmail.com
姓名: 王小明
角色: 學生

Email: student2.gas.course@gmail.com
姓名: 李小華
角色: 學生

Email: student3.gas.course@gmail.com
姓名: 張小美
角色: 學生
```

#### 3. 審核者帳號（審核人員）× 1

**用途：**
- 測試審核流程
- 權限管理測試
- 通知郵件測試

**帳號資訊範例：**
```
Email: reviewer.gas.course@gmail.com
姓名: 陳主任
角色: 審核者
```

### 帳號數量建議

| 課程規模 | 教師帳號 | 學生帳號 | 審核者帳號 | 總計 |
|---------|---------|---------|-----------|------|
| 小型（<20人） | 1 | 3 | 1 | 5 |
| 中型（20-40人） | 1 | 5 | 1 | 7 |
| 大型（>40人） | 2 | 5 | 2 | 9 |

## 帳號建立步驟

### 方法 1：使用個人 Gmail 帳號

#### 步驟 1：建立 Gmail 帳號

1. 前往 https://accounts.google.com/signup
2. 填寫基本資訊
3. 驗證電話號碼
4. 完成帳號建立

#### 步驟 2：設定帳號資訊

1. 設定顯示名稱（如：張老師、王小明）
2. 上傳大頭照（選用）
3. 設定安全性選項

#### 步驟 3：記錄帳號資訊

建立一個試算表記錄所有測試帳號：

| 角色 | Email | 密碼 | 姓名 | 用途 | 狀態 |
|------|-------|------|------|------|------|
| 教師 | teacher@... | *** | 張老師 | 管理者 | 啟用 |
| 學生 | student1@... | *** | 王小明 | 測試 | 啟用 |

**重要：** 密碼應妥善保管，不要公開分享。

### 方法 2：使用 Google Workspace（學校帳號）

如果學校有 Google Workspace，可以使用學校帳號：

#### 優點
- 統一管理
- 不需額外建立帳號
- 符合學校政策

#### 注意事項
- 確認學校政策允許用於測試
- 注意資料隱私
- 課程結束後清理測試資料

## 範本檔案建立

### 建立順序

建議按照以下順序建立範本檔案：

1. 學生成績試算表範本
2. 通知名單試算表範本
3. 簽到記錄試算表範本
4. 服務申請系統試算表範本
5. 成績單文件範本
6. 簽到表單
7. 服務申請表單

### 詳細建立步驟

#### 1. 建立學生成績試算表範本

**步驟：**

1. 使用教師帳號登入 Google Drive
2. 建立新試算表，命名為「學生成績範本」
3. 建立標題列：

```
| 學號 | 姓名 | 國文 | 英文 | 數學 | 平均 | 等級 | 更新時間 |
```

4. 設定標題列格式：
   - 粗體
   - 背景色：#4A86E8（藍色）
   - 文字顏色：白色

5. 填入範例資料（至少 5 筆）：

```
S20240001  王小明  85  90  88  87.67  優  2024-01-15 10:30:00
S20240002  李小華  78  82  75  78.33  良  2024-01-15 10:31:00
S20240003  張小美  92  95  90  92.33  優  2024-01-15 10:32:00
S20240004  陳小強  65  70  68  67.67  可  2024-01-15 10:33:00
S20240005  林小芳  55  58  52  55.00  待加強  2024-01-15 10:34:00
```

6. 設定條件格式：
   - 選取成績欄位（C2:E）
   - 格式 > 條件格式
   - 條件 1：值 >= 60，背景色 #B7E1CD（淺綠色）
   - 條件 2：值 < 60，背景色 #F4C7C3（淺紅色）

7. 記錄檔案 ID（從網址列複製）

#### 2. 建立通知名單試算表範本

**步驟：**

1. 建立新試算表，命名為「通知名單範本」
2. 建立標題列：

```
| 姓名 | Email | 通知類型 | 狀態 | 最後寄送時間 | 備註 |
```

3. 填入範例資料：

```
王小明  wang@school.edu    成績通知  待寄送  -                    期中考成績
李小華  lee@school.edu     活動通知  已寄送  2024-01-15 10:30:00  校慶活動
張小美  chang@school.edu   繳費通知  待寄送  -                    學費繳納
```

4. 記錄檔案 ID

#### 3. 建立簽到記錄試算表範本

**步驟：**

1. 建立新試算表，命名為「簽到記錄範本」
2. 建立工作表 1：簽到記錄

```
| 日期 | 時間 | 姓名 | Email | 狀態 | 備註 |
```

3. 建立工作表 2：每日統計

```
| 日期 | 總人數 | 準時人數 | 遲到人數 | 準時率 |
```

4. 填入範例資料
5. 記錄檔案 ID

#### 4. 建立服務申請系統試算表範本

**步驟：**

1. 建立新試算表，命名為「服務申請系統範本」
2. 建立工作表 1：申請記錄

```
| 申請編號 | 申請日期 | 申請人 | Email | 服務類型 | 說明 | 狀態 | 審核人 | 審核日期 | 審核意見 |
```

3. 建立工作表 2：服務類型

```
| 服務代碼 | 服務名稱 | 負責人 Email | 說明 |
equipment  設備借用  admin@school.edu  借用教學設備
venue      場地申請  venue@school.edu  申請教室或場地
repair     維修申請  repair@school.edu 設備維修申請
```

4. 建立工作表 3：統計報表

```
| 月份 | 服務類型 | 申請數 | 通過數 | 拒絕數 | 處理中 |
```

5. 記錄檔案 ID

#### 5. 建立成績單文件範本

**步驟：**

1. 建立新 Google Docs，命名為「成績單範本」
2. 輸入以下內容：

```
================================
        學生成績單
================================

學號：{{學號}}
姓名：{{姓名}}
班級：{{班級}}

--------------------------------
科目成績
--------------------------------

國文：{{國文}} 分
英文：{{英文}} 分
數學：{{數學}} 分

--------------------------------
統計資料
--------------------------------

平均分數：{{平均}} 分
成績等級：{{等級}}
班級排名：{{排名}}

--------------------------------

發放日期：{{日期}}
```

3. 設定格式（置中、適當字體大小）
4. 記錄檔案 ID

#### 6. 建立簽到表單

**步驟：**

1. 前往 Google Forms（https://forms.google.com）
2. 建立新表單，命名為「每日簽到表單」
3. 設定說明：「請填寫以下資訊完成簽到」
4. 新增題目：

**題目 1：姓名**
- 類型：簡答
- 必填：是

**題目 2：Email**
- 類型：簡答
- 必填：是
- 驗證：電子郵件

**題目 3：備註**
- 類型：簡答
- 必填：否

5. 設定 > 回應
   - 收集電子郵件地址：是
   - 限制為 1 次回應：否
   - 允許回應編輯：否

6. 連結到試算表：
   - 回應 > 建立試算表
   - 選擇「建立新試算表」
   - 命名為「簽到表單回應」

7. 記錄表單 ID 和回應試算表 ID

#### 7. 建立服務申請表單

**步驟：**

1. 建立新表單，命名為「校內服務申請表單」
2. 設定說明：「請填寫以下資訊提交申請」
3. 新增題目：

**題目 1：姓名**
- 類型：簡答
- 必填：是

**題目 2：Email**
- 類型：簡答
- 必填：是
- 驗證：電子郵件

**題目 3：服務類型**
- 類型：單選
- 必填：是
- 選項：設備借用、場地申請、維修申請

**題目 4：申請說明**
- 類型：段落
- 必填：是
- 說明：請詳細說明您的需求

**題目 5：預計使用日期**
- 類型：日期
- 必填：是

4. 連結到試算表
5. 記錄表單 ID 和回應試算表 ID

## 測試資料準備

### 測試資料類型

#### 1. 正常資料
- 用於測試基本功能
- 符合所有驗證規則
- 涵蓋各種情境

#### 2. 邊界資料
- 測試極限值
- 空值、最大值、最小值
- 特殊字元

#### 3. 錯誤資料
- 測試錯誤處理
- 格式錯誤
- 缺少必填欄位

### 範例測試資料集

#### 學生成績測試資料

```javascript
// 正常資料
var normalData = [
  ['S20240001', '王小明', 85, 90, 88],
  ['S20240002', '李小華', 78, 82, 75],
  ['S20240003', '張小美', 92, 95, 90]
];

// 邊界資料
var boundaryData = [
  ['S20240004', '陳小強', 0, 0, 0],      // 最低分
  ['S20240005', '林小芳', 100, 100, 100], // 最高分
  ['S20240006', '黃小明', 60, 60, 60]    // 及格邊界
];

// 錯誤資料（用於測試驗證）
var errorData = [
  ['', '無學號', 85, 90, 88],           // 缺少學號
  ['S20240007', '', 85, 90, 88],        // 缺少姓名
  ['S20240008', '王大明', -10, 90, 88], // 負分
  ['S20240009', '李大華', 85, 150, 88]  // 超過100分
];
```

#### Email 測試資料

```javascript
// 正常 Email
var validEmails = [
  'wang@school.edu',
  'lee@school.edu',
  'chang@school.edu'
];

// 錯誤 Email（用於測試驗證）
var invalidEmails = [
  'invalid-email',      // 缺少 @
  '@school.edu',        // 缺少使用者名稱
  'wang@',              // 缺少網域
  'wang @school.edu'    // 包含空格
];
```

## 權限設定

### 範本檔案權限設定

#### 1. 試算表範本

**建議權限：**
- 知道連結的任何人都可以「檢視」
- 不要設定為「編輯」，避免被誤改

**設定步驟：**
1. 開啟試算表
2. 點選右上角「共用」
3. 點選「變更為知道連結的任何人」
4. 選擇「檢視者」
5. 複製連結

#### 2. 文件範本

**建議權限：**
- 知道連結的任何人都可以「檢視」

**設定步驟：**
同試算表

#### 3. 表單

**建議權限：**
- 任何人都可以填寫（不需登入）
- 或限制為組織內部（如果使用學校帳號）

**設定步驟：**
1. 開啟表單
2. 設定 > 回應
3. 選擇「限制為 1 次回應」（需要登入）
4. 或取消勾選（不需登入）

### 測試帳號權限設定

#### 給予學生帳號編輯權限

某些測試情境需要給予學生帳號編輯權限：

1. 開啟試算表
2. 點選「共用」
3. 輸入學生帳號 Email
4. 選擇「編輯者」
5. 取消勾選「通知使用者」
6. 點選「共用」

#### Google Sheets / Docs
從網址列複製：

#### Google Forms
從網址列複製：
```
https://docs.google.com/forms/d/[這裡是ID]/edit
```

## 常見問題

### Q1: 測試帳號被鎖定怎麼辦？

A: 
1. 檢查是否因為異常登入被 Google 鎖定
2. 使用帳號復原流程
3. 驗證電話號碼
4. 如無法復原，建立新帳號

### Q2: 範本檔案被誤刪怎麼辦？

A: 
1. 到 Google Drive 垃圾桶還原
2. 如已清空垃圾桶，從備份還原
3. 重新建立範本

### Q3: 學習者無法存取範本？

A: 
1. 檢查權限設定
2. 確認連結正確
3. 檢查學習者帳號是否正常
4. 嘗試重新分享

### Q4: 表單回應沒有記錄到試算表？

A: 
1. 檢查表單是否正確連結試算表
2. 檢查試算表權限
3. 重新連結表單與試算表
4. 測試提交表單
