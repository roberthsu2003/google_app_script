# 單元 1：認識 Apps Script 與 JavaScript 基礎

**時數：** 10 小時（理論 4 小時 + 實作 6 小時）

## 學習目標

- 了解 Google Apps Script 的特性與應用場景
- 掌握 JavaScript 基礎語法
- 理解 GAS 對 JavaScript 語法的支援情況
- 能夠撰寫簡單的 GAS 腳本

## 教學內容

### 1.1 認識 Google Apps Script（1 小時）
- GAS 簡介與應用場景
- GAS 編輯器介面導覽
- 第一個 Hello World 程式
- V8 runtime 介紹與啟用方式

### 1.2 JavaScript 變數與資料型別（1.5 小時）
- 變數宣告：var、let、const
- 基本資料型別
- 運算子
- 10 個練習題

### 1.3 流程控制（2 小時）
- if-else 條件判斷
- switch-case 語法
- 迴圈（for、while、do-while）
- 15 個練習題

### 1.4 函式（1.5 小時）
- 函式定義與呼叫
- 參數與回傳值
- 作用域
- 10 個練習題

### 1.5 陣列操作（1.5 小時）
- 陣列基本操作
- 常用陣列方法
- 陣列遍歷
- 12 個練習題

### 1.6 物件與字串（1.5 小時）
- 物件操作
- JSON 處理
- 字串處理
- 18 個練習題

### 1.7 錯誤處理與除錯（1 小時）
- try-catch-finally
- Logger.log() 除錯
- 5 個實作範例
