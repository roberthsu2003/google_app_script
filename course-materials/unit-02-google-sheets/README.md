# 單元 2：操作 Google Sheets

**時數：** 6 小時（理論 2 小時 + 實作 4 小時）

## 學習目標

- 掌握 SpreadsheetApp 服務的核心方法
- 能夠讀取、寫入、格式化試算表資料
- 理解範圍（Range）操作的概念
- 完成學生成績登錄系統實作

## 教學內容

### 2.1 SpreadsheetApp 基礎（1 小時）
- SpreadsheetApp 服務架構
- 取得試算表
- 工作表操作
- 範圍選取

### 2.2 資料讀取與寫入（1.5 小時）
- getValue() / getValues()
- setValue() / setValues()
- 批次操作效能優化

### 2.3 格式化與進階操作（1.5 小時）
- 格式設定
- 排序與篩選
- 公式操作

### 2.4 實作案例 1：學生成績登錄系統（2 小時）
- 成績輸入介面
- 自動計算平均分數
- 等級判定
- 自動格式化
