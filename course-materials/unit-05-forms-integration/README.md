# 單元 5：表單整合

**時數：** 4 小時（理論 1.5 小時 + 實作 2.5 小時）

## 學習目標

- 深入掌握 FormApp 服務
- 能夠程式化建立與管理表單
- 理解表單觸發器的應用
- 完成表單回應自動處理系統

## 教學內容

### 5.1 FormApp 進階操作（1.5 小時）
- 程式化建立表單
- 新增題目
- 設定題目屬性
- 取得表單回應

### 5.2 表單觸發器與自動化（1 小時）
- 表單提交觸發器
- 觸發器事件物件
- 即時處理表單回應

### 5.3 實作案例 6：簡易簽到系統（1.5 小時）
- 建立簽到表單
- 自動記錄到試算表
- 發送確認郵件
- 產生統計報表
