# 單元 6：建立簡易 Web App

**時數：** 8 小時（理論 3 小時 + 實作 5 小時）

## 學習目標

- 掌握 HTML Service 建立 Web 應用程式
- 理解前端與後端的互動機制
- 能夠設計簡單的使用者介面
- 完成兩個 Web App 實作專案

## 教學內容

### 6.1 HTML Service 基礎（1.5 小時）
- HTML Service 簡介
- 建立 HTML 檔案
- 部署 Web App
- 權限設定

### 6.2 前後端互動（2 小時）
- google.script.run
- 成功與失敗回調
- 參數傳遞
- 非同步處理

### 6.3 HTML/CSS/JavaScript 基礎（1.5 小時）
- HTML 基本標籤
- CSS 基本樣式
- JavaScript DOM 操作
- Bootstrap 使用

### 6.4 實作案例 7：每日報表寄送機器人（1.5 小時）
- Web 介面設定
- 資料提取
- 報表產生
- 定時寄送

### 6.5 實作案例 8：專案檔案上傳管理工具（1.5 小時）
- 檔案上傳介面
- 檔案管理
- 記錄管理
- 清單顯示
