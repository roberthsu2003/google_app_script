# 實作案例 7：每日報表寄送機器人

## 專案概述

建立一個每日報表寄送機器人，透過 Web 介面設定報表參數，自動從試算表提取資料、產生 HTML 格式報表，並定時寄送給指定收件者。

## 學習目標

- 整合 Web App 與試算表資料
- 學習 HTML 表格格式化
- 掌握定時觸發器設定
- 練習郵件內容美化
- 實作參數設定介面

## 功能需求

### 核心功能

1. **Web 設定介面**
   - 選擇資料來源（試算表）
   - 設定收件者
   - 選擇報表類型
   - 設定寄送時間

2. **資料提取**
   - 從試算表讀取資料
   - 資料篩選與統計
   - 格式化處理

3. **報表產生**
   - HTML 表格格式
   - 圖表統計（選擇性）
   - 美化樣式

4. **自動寄送**
   - 定時觸發器
   - 手動觸發功能
   - 寄送記錄

## 技術實作

詳見 `Code.gs` 和 `implementation-guide.md`

## 使用說明

1. 執行 `setup()` 初始化系統
2. 開啟 Web App 設定報表參數
3. 測試報表產生
4. 設定定時觸發器
