# 實作案例 8：專案檔案上傳管理工具

## 專案概述

建立一個檔案上傳管理工具，讓使用者透過 Web 介面上傳檔案到 Google Drive，並自動記錄上傳資訊到試算表。

## 學習目標

- 實作檔案上傳功能
- 整合 DriveApp 與 Web App
- 學習檔案清單顯示
- 掌握下載連結產生
- 練習前後端資料傳遞

## 功能需求

### 核心功能

1. **檔案上傳介面**
   - 選擇檔案
   - 上傳進度顯示
   - 上傳成功提示

2. **檔案管理**
   - 儲存到指定資料夾
   - 記錄上傳者與時間
   - 產生分享連結

3. **檔案清單**
   - 顯示已上傳檔案
   - 提供下載連結
   - 檔案資訊顯示

4. **記錄管理**
   - 寫入試算表
   - 查詢功能
   - 統計資訊

## 技術實作

詳見 `Code.gs` 和 `implementation-guide.md`

## 使用說明

1. 執行 `setup()` 初始化系統
2. 開啟 Web App
3. 選擇並上傳檔案
4. 查看檔案清單
