# 單元 8：綜合實作展示

**時數：** 10 小時（理論 2 小時 + 實作 8 小時）

## 學習目標

- 整合所有已學技能
- 完成一個完整的綜合專案
- 學習專案規劃與開發流程
- 展示與分享作品

## 教學內容

### 8.1 專案規劃與需求分析（1 小時）
- 專案開發流程
- 需求分析方法
- 功能拆解
- 時程規劃

### 8.2 實作案例 10：校內服務申請系統（7 小時）
- 申請表單開發
- 資料記錄與郵件通知
- 審核管理介面
- 統計報表功能
- 測試與除錯

### 8.3 專案展示與分享（1 小時）
- 展示專案功能
- 說明技術實作
- 分享開發心得
- 同儕回饋

### 8.4 作品集建立指引（1 小時）
- 整理專案文件
- 撰寫專案說明
- 建立作品集清單

## 資料夾結構

```
unit-08-final-project/
├── README.md (本檔案)
├── lectures/
├── case-study-10/ (實作案例 10)
└── portfolio-guide/ (作品集指引)
```

## 單元產出

- 完成校內服務申請系統（實作案例 10）
- 建立個人作品集（包含 10 個完整專案）
